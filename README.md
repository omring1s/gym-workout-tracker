# 🏋️ Gym Workout Tracker

A professional, interactive web app for tracking gym workouts.  
Log exercises by body part, track your weight and reps, visualize progress with dynamic charts, and maintain a history of all workouts.

## **Features**

- ✅ Log workouts by **body part**, **exercise**, **sets**, **reps**, **weight**, **date**, and **notes**  
- ✅ Add **custom exercises** in addition to preset exercises  
- ✅ Log **past workouts** with a date picker  
- ✅ **Edit or delete** any workout entry  
- ✅ **Interactive charts** for visualizing progress  
- ✅ **Filter workouts** by body part or exercise  
- ✅ Responsive, clean, and user-friendly UI  


## **Tech Stack**

- **Frontend:** HTML, CSS, JavaScript, Chart.js  
- **Backend:** Node.js, Express  
- **Persistence:** JSON file (`workouts.json`)  

## **Installation**

1. **Clone the repository:**

git clone https://github.com/YOUR_USERNAME/gym-workout-tracker.git

cd gym-workout-tracker

2. **Install backend dependencies**

cd server
npm install express cors

3. **Start the backend server**

node server.js

4. **Open the frontend**

Open client/index.html in your browser.
